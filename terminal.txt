===
Installasi dan Konfigurasi Tailwind CSS di Laravel
===
1. Installasi Tailwind CSS
   "Sejak Laravel 12 dirilis, Tailwind CSS sudah otomatis terinstall di dalam project,
    maka kita tidak perlu melakukan Installasi secara manual lagi."
2. Menambahkan Plugin Form
   > npm install -D @tailwindcss/forms@0.5.10
3. Compile Tailwind CSS dengan Vite
   > npm run dev

===
Membuat Model dan Migration
===
1. Category
    > php artisan make:model Category -m
2. Campaign
    > php artisan make:model Campaign -m
3. Donatur
    > php artisan make:model Donatur -m
4. Donation
    > php artisan make:model Donation -m
5. Slider
    > php artisan make:model Slider -m

Jalankan Migration
> php artisan migrate


===
Membuat Data Seeder User
===
> php artisan make:seeder UserSeeder
> php artisan db:seed --class=UserSeeder

===
Installasi dan Konfigurasi Laravel Fortify
===
1. Installasi
    > composer require laravel/fortify:1.25.4
2. Publish Konfigurasi
    > php artisan fortify:install
    > php artisan migrate

===
Membuat CRUD Category
===
1. Membuat Controller Category
    > php artisan make:controller Admin/CategoryController
2. Melihat route list
    > php artisan route:list
3. Membuat view/halaman index Category
    catatan: Untuk menampilkan data perhalaman, menggunakan pagination, di dalam Laravel sudah disediakan template untuk pagination,
         baik itu Bootstrap, Semantic UI, Tailwind CSS dan lain-lain. Oleh karena itu, kita cukup melakukan publish view/template
         yang sudah disediakan tersebut.
    Silahkan jalankan perintah berikut ini di dalam terminal/CMD :
    > php artisan vendor:publish --tag=laravel-pagination
